# Intranet ITRC

Intranet institucional de la Agencia del Inspector General de Tributos, Rentas y Contribuciones Parafiscales.

Astro 4 + Sveltia CMS. Desplegado en GitHub Pages.

## Comandos

```
npm install        # instalar dependencias
npm run dev        # servidor local (localhost:4321)
npm run build      # compilar para produccion
```

## CMS

Acceso en `/admin`. Configuracion generada desde `cms/`.
